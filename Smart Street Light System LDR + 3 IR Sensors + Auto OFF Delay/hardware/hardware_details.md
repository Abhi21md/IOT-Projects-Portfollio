HARDWARE DETAILS
----------------

1. Arduino UNO
   - Main controller

2. LDR Sensor
   - Connected to A0
   - Detects ambient light

3. IR Sensors
   - IR1 → Pin 2
   - IR2 → Pin 3
   - IR3 → Pin 4
   - Detect vehicle movement

4. Street Lamps (LEDs / Relays)
   - Lamp1 → Pin 5
   - Lamp2 → Pin 6
   - Lamp3 → Pin 7

5. Power Supply
   - 5V regulated supply

PIN CONNECTION TABLE:

LDR      → A0
IR1      → D2
IR2      → D3
IR3      → D4
LAMP1    → D5
LAMP2    → D6
LAMP3    → D7

NOTE:
If using real street lamps, use relay modules instead of direct LEDs.
