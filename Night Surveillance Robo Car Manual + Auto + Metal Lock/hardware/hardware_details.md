# Hardware Details

Board: NodeMCU (ESP8266)
Motor Driver: L298N
Motors: 2 DC Motors
Power: 7.4V / 12V battery
Platform: Blynk IoT

--------------------------------------------------

## Pin Configuration

ENA → D0
IN1 → D1
IN2 → D2
IN3 → D3
IN4 → D4
ENB → D5

--------------------------------------------------

## Wiring Summary

NodeMCU → L298N

D0 → ENA
D1 → IN1
D2 → IN2
D3 → IN3
D4 → IN4
D5 → ENB

Battery → L298N power input
Motors → Motor A & B outputs

--------------------------------------------------

## Required Components

• NodeMCU ESP8266
• L298N Motor Driver
• 2 DC Motors
• Robot chassis
• Battery pack
• Jumper wires
• Blynk mobile app
