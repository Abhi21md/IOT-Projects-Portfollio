# Night Surveillance Robo Car (ESP8266)

An IoT-based WiFi surveillance robot with manual and auto patrol mode.

------------------------------------------------

## Features

• Manual remote control via Blynk
• Auto forward-backward patrol
• Adjustable speed
• Time-based movement
• Stable WiFi connectivity

------------------------------------------------

## Folder Structure

config/      → WiFi & Blynk credentials  
firmware/    → Main Arduino code  
docs/        → System documentation  
hardware/    → Wiring and components  
test/        → Testing reports  

------------------------------------------------

## Modes

Manual Mode:
Control robot using mobile app buttons.

Auto Mode:
Robot moves forward and backward continuously based on user-defined time.
