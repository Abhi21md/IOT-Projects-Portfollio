# Dual MCU Night Surveillance Robo System

An advanced IoT-based surveillance rover combining:

• Blynk controlled mobile robot
• Metal detection safety lock
• Dual ultrasonic radar defense system
• Laser alert mechanism
• ESP32-CAM live streaming

---

## System Architecture

NodeMCU → Motor + Metal Safety  
Radar MCU → Dual Sensor Tracking + Laser  
ESP32-CAM → Live Video Stream  

---

## Features

✔ Manual control via mobile  
✔ Auto patrol mode  
✔ Metal detection safety stop  
✔ Dual-side radar detection  
✔ Laser beam target alert  
✔ Live camera streaming  

---

## Technology Used

- ESP8266 (NodeMCU)
- ESP32-CAM
- Ultrasonic Sensors
- Metal Detector Module
- Servo Motors
- L298N Motor Driver
- Blynk IoT Platform
