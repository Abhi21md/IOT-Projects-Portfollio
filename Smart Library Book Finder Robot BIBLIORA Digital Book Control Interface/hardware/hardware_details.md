HARDWARE CONNECTION DETAILS
----------------------------

Board: NodeMCU (ESP8266)

Motor Driver (L298N):

IN1 → D1 (GPIO5)
IN2 → D2 (GPIO4)
IN3 → D6 (GPIO12)
IN4 → D7 (GPIO13)
ENA → D5 (GPIO14)
ENB → D8 (GPIO15)

IR Sensor:
OUT → D0 (GPIO16)
VCC → 3.3V
GND → GND

Buzzer:
Positive → D3 (GPIO0)
Negative → GND

LED Indicator:
Positive → D4 (GPIO2)
Negative → GND

Power:
External battery recommended for motors.

NOTE:
Ensure common GND between NodeMCU and motor driver.
