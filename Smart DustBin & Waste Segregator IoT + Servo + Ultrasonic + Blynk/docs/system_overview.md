Smart Dustbin & Waste Segregator (IoT Based)

This project automatically detects and segregates waste into Dry and Wet categories using:

- Ultrasonic presence detection
- Soil moisture sensor for waste type
- Servo motor for automatic segregation
- Dual ultrasonic sensors for level monitoring
- NodeMCU ESP8266 for IoT connectivity
- Blynk Cloud integration

Features:
- Automatic dry/wet detection
- Real-time bin level monitoring
- LCD status display
- 75% full alert notification
- Blynk email + event notification
- Cloud monitoring dashboard
