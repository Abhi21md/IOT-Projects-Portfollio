# Smart EV Wireless Charging Station

An IoT-based intelligent EV charging system with ultrasonic parking detection and automatic timer control.

--------------------------------------------------

## Core Features

• Automatic charging start at perfect parking
• Manual override via Blynk
• Timer-based auto cutoff
• Non-blocking buzzer scheduler
• Robust ultrasonic multi-sampling
• LCD live monitoring
• Cloud-based control & status

--------------------------------------------------

## Technology Stack

Board: NodeMCU ESP8266
Platform: Blynk IoT
Sensor: Ultrasonic (HC-SR04)
Display: I2C LCD
Alert: Active buzzer
Control: Relay module

--------------------------------------------------
