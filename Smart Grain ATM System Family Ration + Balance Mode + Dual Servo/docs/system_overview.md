# Smart Grain ATM System

Board: ESP32
Technology: RFID + HX711 Load Cell + Servo Motors + Blynk IoT

------------------------------------------------------

## Project Overview

This system is a smart automated grain dispensing ATM that supports:

1. Paid Balance Mode (like digital wallet)
2. Family Ration Mode (government ration model)
3. Daily ration control system
4. IoT cloud monitoring dashboard

------------------------------------------------------

## Two Operating Modes

### 1️⃣ Balance Mode (Ajit)

• User has prepaid balance
• Select 200g or 250g
• Select Rice or Jowar
• System deducts money
• Grain dispensed using weight sensor feedback

### 2️⃣ Family Ration Mode (Fam1 & Fam2)

• Predefined family members
• 100g rice per head
• 50g jowar per head
• Only once per 24 hours
• Auto daily reset
