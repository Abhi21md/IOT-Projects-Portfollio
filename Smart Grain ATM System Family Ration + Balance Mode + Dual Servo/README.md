# Smart Grain ATM System

An IoT-based automated grain distribution system
supporting paid and government-style family ration models.

------------------------------------------------------

## Core Features

• RFID user authentication
• Digital balance deduction system
• 200g / 250g quantity selection
• Rice and Jowar dispensing
• Load cell weight verification
• Daily ration lock system
• 24-hour automatic reset
• IoT cloud monitoring via Blynk

------------------------------------------------------

## Why This Project Is Advanced

Unlike simple vending machines:

✔ Supports two distribution models
✔ Tracks family-based ration limits
✔ Performs real-time weight verification
✔ Prevents duplicate daily claims
✔ Tracks financial collection
✔ Tracks total grain distributed
✔ Cloud-connected analytics

• Government ration shops
