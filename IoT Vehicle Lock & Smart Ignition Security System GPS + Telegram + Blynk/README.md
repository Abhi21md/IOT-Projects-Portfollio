# Smart Vehicle Lock & Ignition Security System

An IoT-based anti-theft vehicle security system with GPS tracking
and real-time Telegram alert integration.

--------------------------------------------------

## Key Features

• Remote lock/unlock control
• Remote ignition control
• Physical ignition button
• Ignition blocking when locked
• Real-time GPS tracking
• Telegram alert with Google Maps link
• Smart GPS fallback system
• LCD + Cloud display integration

--------------------------------------------------

## Why This Project Is Advanced

Unlike normal vehicle lock systems:

✔ Prevents unauthorized ignition
✔ Sends real-time location alerts
✔ Uses dual control (cloud + physical)
✔ Has backup GPS fallback logic
✔ Integrates Telegram security alerting

--------------------------------------------------

## Applications

• Two-wheelers
• Cars
• Fleet security
• Anti-theft systems
• Smart vehicle prototypes

--------------------------------------------------

Technology Stack:
ESP8266 + GPS NEO-6M + Blynk + Telegram API + Relay Control
