# Smart Voltage & Current Monitoring System (IoT)

An ESP32 based real-time AC protection system with Blynk IoT integration.

## Features
- RMS Voltage Measurement
- RMS Current Measurement
- Over/Under Voltage Protection
- Over Current Protection
- Automatic Relay Cutoff
- Buzzer Alert System
- Manual Relay Override
- Live Blynk Monitoring
- Auto Zero Calibration

## Technology Used
- ESP32
- ZMPT101B
- ACS712
- Blynk IoT
- I2C LCD

## Folder Structure
config/      → Credentials & configuration  
docs/        → System documentation  
firmware/    → Source code  
hardware/    → Wiring & components  
test/        → Test reports
