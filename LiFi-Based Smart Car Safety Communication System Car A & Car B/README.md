# LiFi-Enabled Smart Car Safety Communication System

An advanced IoT vehicle safety system combining:

• WiFi Control (Blynk)
• Smoke Detection Lock
• LiFi Communication
• Accident & Alcohol Pattern Detection

--------------------------------------------------

## System Modules

Car-A:
WiFi controlled vehicle with MQ3 smoke lock and LiFi transmitter.

Car-B:
LiFi receiver that decodes frequency patterns and displays safety alerts.

--------------------------------------------------

## Key Innovations

• Dual-layer safety system
• Wireless WiFi + Optical LiFi communication
• Frequency-based pattern decoding
• Confirmation-based accident detection
• Event logging via Blynk

--------------------------------------------------

## Folder Structure

config/      → WiFi & Blynk credentials  
firmware/    → Car-A and Car-B codes  
docs/        → System explanation
