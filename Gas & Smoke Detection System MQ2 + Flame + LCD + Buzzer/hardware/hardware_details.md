HARDWARE CONNECTIONS
--------------------

Arduino UNO

----------- MQ2 -----------
VCC → 5V
GND → GND
AO  → A0

----------- FLAME SENSOR -----------
VCC → 5V
GND → GND
DO  → Pin 6

----------- BUZZER -----------
Positive → Pin 8
Negative → GND

----------- ALERT LED -----------
Anode → Pin 9
Cathode → GND (via 220Ω resistor)

----------- LCD I2C -----------
SDA → A4
SCL → A5
VCC → 5V
GND → GND
