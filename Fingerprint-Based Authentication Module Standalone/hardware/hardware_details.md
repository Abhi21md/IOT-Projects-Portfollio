HARDWARE CONNECTION DETAILS
----------------------------

Fingerprint Sensor:
----------------------------
Sensor TX → Arduino Pin 2
Sensor RX → Arduino Pin 3
VCC → 5V
GND → GND

LCD (16x2 I2C):
----------------------------
SDA → A4 (UNO)
SCL → A5 (UNO)
VCC → 5V
GND → GND

Relay Module:
----------------------------
IN → Pin 7
VCC → 5V
GND → GND

Buzzer:
----------------------------
Positive → Pin 6
Negative → GND

Power Supply:
----------------------------
5V regulated power supply
Common ground required for all modules
