# Hardware Details

-----------------------------------
Microcontroller
-----------------------------------
ESP32 Dev Module

-----------------------------------
Motor Driver
-----------------------------------
L298N or compatible H-Bridge

-----------------------------------
Pin Connections

Motor A:
ENA → GPIO 16
IN1 → GPIO 17
IN2 → GPIO 18

Motor B:
IN3 → GPIO 19
IN4 → GPIO 21
ENB → GPIO 22

-----------------------------------
Power Supply
-----------------------------------
• 7.4V – 12V battery for motors
• 5V regulated supply for ESP32

-----------------------------------
Required Components

• ESP32 Dev Board
• L298N Motor Driver
• 2 DC Motors
• Chassis + Wheels
• Battery Pack
• Jumper Wires
