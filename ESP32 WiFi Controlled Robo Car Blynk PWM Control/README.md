# ESP32 WiFi Controlled Robo Car

An IoT based smart robo car controlled through Blynk mobile application.

--------------------------------------------------

## Features

• WiFi based remote control
• Real-time direction control
• Speed control via slider
• Hardware PWM using ESP32 LEDC
• Simple and stable logic
• Serial debugging support

--------------------------------------------------

## Technology Used

Board: ESP32  
Platform: Blynk IoT  
Language: Arduino C++  
Motor Control: H-Bridge Driver  
Communication: WiFi  

--------------------------------------------------

## Application Areas

• Robotics learning
• IoT demonstrations
• Smart vehicle prototypes
• Educational robotics labs
