# IoT Water Quality Monitoring System

Real-time water quality monitoring using NodeMCU,
TDS sensor, DS18B20 temperature sensor, and Blynk IoT.

---

## Features

• Real-time TDS monitoring (ppm)
• Water temperature monitoring
• Cloud dashboard integration
• Buzzer alert system
• Automatic Blynk notification
• Compact and low-cost design

---

## Hardware Used

- NodeMCU (ESP8266)
- DS18B20 Temperature Sensor
- TDS Sensor Module
- Buzzer

---

## Blynk Dashboard Setup

V0 → Gauge (TDS)
V1 → Gauge (Temperature)

Event Name:
water_alert

---

## Alert Conditions

• TDS > 500 ppm
• Temperature > 32°C

---

## Applications

• RO system monitoring
• Drinking water safety
• Water tank monitoring
• Industrial liquid monitoring
