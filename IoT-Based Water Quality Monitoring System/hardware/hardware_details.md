HARDWARE CONNECTION DETAILS
----------------------------

NodeMCU (ESP8266)

DS18B20 Temperature Sensor:
--------------------------------
VCC → 3.3V
GND → GND
DATA → D2
4.7k resistor between DATA and VCC

TDS Sensor:
--------------------------------
Analog Output → A0
VCC → 3.3V
GND → GND

Buzzer:
--------------------------------
Positive → D1
Negative → GND

Power:
--------------------------------
USB 5V supply to NodeMCU
