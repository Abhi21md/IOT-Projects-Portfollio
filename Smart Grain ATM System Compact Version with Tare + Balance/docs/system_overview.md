SYSTEM OVERVIEW
----------------

The Smart Grain ATM is a standalone automated grain dispensing system
based on RFID authentication and weight-based dispensing.

-----------------------------------
WORKING PRINCIPLE
-----------------------------------

1) User scans RFID card.
2) System verifies UID.
3) LCD displays user balance.
4) User selects quantity (100g / 200g).
5) HX711 measures real-time weight.
6) Servo opens gate for dispensing.
7) When target weight reached:
      → Gate closes
      → Balance deducted
      → LCD shows updated balance

-----------------------------------
SAFETY FEATURES
-----------------------------------

• Invalid card rejection
• Low balance protection
• Manual tare calibration button
• Real-time weight display
• Servo auto close after dispensing

-----------------------------------
EXPANSION POSSIBILITIES
-----------------------------------

• IoT dashboard integration
• Cloud balance storage
• Daily ration control mode
• Multi-grain selection
