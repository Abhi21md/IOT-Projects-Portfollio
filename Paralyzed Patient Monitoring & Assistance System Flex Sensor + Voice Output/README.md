# Paralyzed Patient Monitoring & Assistance System

A low-cost assistive communication system for paralyzed patients
using flex sensors and voice playback.

--------------------------------------------------

## Key Features

• Gesture-based communication
• 5 predefined voice alerts
• LCD message display
• DFPlayer Mini audio playback
• Simple and reliable logic
• No internet required

--------------------------------------------------

## Applications

• Hospitals
• Home patient care
• Elderly assistance
• Rehabilitation centers

--------------------------------------------------

## Technology Stack

Microcontroller: Arduino Nano
Sensors: Flex Sensors
Audio Module: DFPlayer Mini
Display: I2C LCD
