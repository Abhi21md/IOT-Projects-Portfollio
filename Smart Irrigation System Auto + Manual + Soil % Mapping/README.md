# Smart Irrigation System (IoT Based)

An automatic irrigation system using NodeMCU,
soil moisture sensing, and Blynk IoT.

------------------------------------------------------

## Key Features

• Automatic pump control based on soil moisture
• Manual override mode
• Temperature & humidity monitoring
• LCD real-time display
• IoT cloud dashboard monitoring
• HIGH relay logic system

------------------------------------------------------

## How It Works

1. Soil sensor reads moisture level
2. If soil ≥ 90% → Pump ON
3. If soil < 90% → Pump OFF
4. Auto/Manual mode selectable from Blynk
5. Temperature & humidity monitored continuously

------------------------------------------------------

Applications

• Smart Farming
• Garden Automation
• Greenhouse Monitoring
• Water Conservation Systems
• IoT Agriculture Projects

------------------------------------------------------

Technology Used

NodeMCU (ESP8266)
Blynk IoT
DHT11 Sensor
Soil Moisture Sensor
Relay Module
I2C LCD
