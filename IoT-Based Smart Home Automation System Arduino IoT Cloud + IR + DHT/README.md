# Smart Home Automation System (Arduino IoT Cloud)

An ESP32-based IoT smart home system with
motion detection and temperature automation.

------------------------------------------------------

## Key Features

• IR motion-based hall light control
• 5-channel relay switching
• Temperature-triggered appliance control
• Cloud dashboard control (Arduino IoT Cloud)
• Real-time IoT monitoring

------------------------------------------------------

## How It Works

1. IR sensor detects movement → Hall light ON
2. DHT11 monitors temperature
3. If temperature > 30°C → Cooling relay ON
4. All relays can be manually controlled from cloud

------------------------------------------------------

Applications

• Smart Home Systems
• Office Automation
• Energy Saving Systems
• Intelligent Lighting Control
• IoT-based Appliance Monitoring

------------------------------------------------------
