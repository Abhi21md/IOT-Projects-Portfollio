# AI Based Smart Waste Segregation System

An intelligent waste segregation system combining Artificial Intelligence and IoT hardware.

## Features
- Metal Detection
- Moisture Based Wet/Dry Detection
- AI Based Plastic vs Paper Classification
- Stepper Controlled Multi-Bin System
- Servo Gate Mechanism
- Real-time Camera Prediction
- Serial Communication between AI and ESP32

## AI Technology
- PyTorch
- MobileNetV2 Transfer Learning
- Custom Dataset Training

## Hardware Used
- ESP32
- USB Camera
- IR Sensor
- Metal Proximity Sensor
- Soil Moisture Sensor
- Stepper Motor
- Servo Motor
- Buzzer

## Folder Structure
config/        → Credentials  
docs/          → Documentation  
firmware/      → ESP32 + AI code  
hardware/      → Wiring details  
test/          → Test results
