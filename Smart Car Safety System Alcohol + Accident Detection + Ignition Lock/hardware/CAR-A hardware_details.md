# Car-A Hardware (Transmitter)

Board: NodeMCU (ESP8266)

---------------------------------
Alcohol Sensor
---------------------------------
AO → A0

---------------------------------
MPU6050
---------------------------------
SDA → D2
SCL → D1

---------------------------------
Ignition Relay
---------------------------------
Control → D6

---------------------------------
Buzzer
---------------------------------
Pin → D7

---------------------------------
Alert LED (LiFi Transmitter)
---------------------------------
Pin → D5
(High power LED recommended)

---------------------------------
Blynk Virtual Pins
---------------------------------
V3 → Alcohol Value
V4 → Ignition Status
V5 → LCD Widget
V6 → Threshold Control
