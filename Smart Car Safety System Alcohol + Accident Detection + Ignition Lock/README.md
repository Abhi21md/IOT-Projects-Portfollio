# LiFi Based Smart Car Safety Communication System

A dual-vehicle intelligent safety system using light pulse communication.

Car-A detects accident or alcohol and transmits encoded light patterns.

Car-B receives and decodes the signal to alert the front vehicle.

------------------------------------------------------------

Key Features

• Accident detection using MPU6050
• Alcohol detection with ignition lock
• Light-based communication (LiFi)
• Frequency-based pattern encoding
• LCD based alert system
• Blynk cloud logging
• Continuous and timed alert modes

------------------------------------------------------------

Why This Project Is Advanced

• Dual microcontroller architecture
• Signal encoding & decoding
• Real-time frequency analysis
• Safety-critical event detection
• IoT + Embedded + Communication hybrid

------------------------------------------------------------

Technology Stack

MPU6050
