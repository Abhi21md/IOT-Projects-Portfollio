# Smart LPG Safety System – Fixed Threshold Edition

A compact IoT-based LPG safety system designed for fast detection
and automatic cylinder regulator control.

--------------------------------------------------

## Core Features

• Fixed calibrated gas threshold
• Automatic servo regulator control
• Buzzer alert system
• Custom I2C LCD configuration
• Manual override via Blynk
• Lightweight cloud data handling
• Fast response safety logic

--------------------------------------------------

## Why This Version is Unique

This is a simplified and optimized safety edition:

✔ Faster reaction time  
✔ Minimal cloud dependency  
✔ Stable fixed safety threshold  
✔ Ideal for home deployment  
✔ Custom pin configuration flexibility  

--------------------------------------------------

## Applications

• Residential kitchens  
• Small restaurants  
• LPG storage rooms  
• Safety demo systems  

--------------------------------------------------

Technology Stack:
ESP8266 + MQ2 + Servo + Blynk + LCD
