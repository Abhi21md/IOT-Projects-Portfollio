# IoT-Based Vehicle Speed Monitoring System

A smart traffic monitoring solution using NodeMCU ESP8266.

--------------------------------------------------

## Core Features

• Dual IR sensor speed detection
• Real-time speed calculation
• Adjustable speed limit via Blynk
• Overspeed buzzer alert
• Cloud logging and event notification
• LCD display output
• Car counting system

--------------------------------------------------

## Technology Stack

Microcontroller: ESP8266
Platform: Blynk IoT
Sensors: Infrared obstacle sensors
Display: I2C LCD
Alert: Active HIGH buzzer

--------------------------------------------------

## Application Areas

• Smart highways
• Toll gates
• School zones
• Industrial vehicle monitoring
• Smart city traffic systems
