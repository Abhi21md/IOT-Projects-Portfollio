# Hardware Details

-----------------------------------
Board
-----------------------------------
NodeMCU ESP8266

-----------------------------------
Sensors
-----------------------------------
IR1 → D5
IR2 → D6

-----------------------------------
Output Devices
-----------------------------------
Buzzer (Active HIGH) → D7
LCD 16x2 I2C → SDA, SCL

-----------------------------------
Connections
-----------------------------------
Distance between IR sensors = 20 cm (0.20 meters)

Power:
5V regulated supply or battery pack

-----------------------------------
Required Components
-----------------------------------
• NodeMCU ESP8266
• 2 IR obstacle sensors
• 16x2 I2C LCD
• Active HIGH buzzer
• Jumper wires
• Breadboard
