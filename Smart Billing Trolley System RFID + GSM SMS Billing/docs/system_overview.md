SMART BILLING TROLLEY SYSTEM (DUAL ESP32)

This project uses two ESP32 microcontrollers.

ESP32 - 1 (Billing Unit):
- RFID based item add/remove
- Load cell grain billing
- Auto calibration
- Invoice generation
- Blynk cloud display

ESP32 - 2 (Auto Follow Unit):
- IR based human tracking
- Ultrasonic distance safety
- Manual Blynk control
- Motor driver based movement

Features:
- Automatic add and remove items
- Weight based billing
- Cloud invoice
- Smart trolley following user
