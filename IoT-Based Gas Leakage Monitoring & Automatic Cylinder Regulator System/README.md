# IoT Gas Leakage Monitoring & Automatic Cylinder Regulator System

An intelligent gas safety system that automatically detects LPG leakage
and closes the cylinder regulator using a servo motor.

--------------------------------------------------

## Key Features

• Real-time gas level monitoring
• Adjustable safety threshold
• Automatic regulator closure
• Manual override control
• Buzzer alert
• LCD real-time display
• Blynk cloud integration
• Reduced cloud spam logic
• Safe actuator control logic

--------------------------------------------------

## Why This System is Advanced

Unlike simple gas detectors, this system:

✔ Automatically shuts the gas supply
✔ Allows remote manual control
✔ Sends real-time alerts
✔ Avoids unnecessary cloud traffic
✔ Provides local + remote monitoring

--------------------------------------------------

## Applications

• Smart homes
• LPG kitchen safety
• Industrial gas monitoring
• Hostel and hotel kitchens
• Gas cylinder storage areas

--------------------------------------------------

## Technology Stack

Microcontroller: ESP8266
Sensor: MQ2 Gas Sensor
Actuator: Servo Motor
Communication: WiFi
Cloud: Blynk IoT
Display: I2C LCD
