# Conveyor Belt Digital Counter System

Interrupt-based industrial counter system
with EEPROM memory and relay motor control.

---

## Features

• High-speed interrupt counting
• EEPROM set value storage
• Start / Stop / Reset control
• Automatic motor stop at target
• Relay output control
• Buzzer alert system
• 16x2 LCD live display

---

## Applications

• Conveyor belt production lines
• Industrial packaging counters
• Bottle counting systems
• Assembly line monitoring
• Manufacturing automation

---

## Technical Highlights

• External interrupt counting (Pin 2)
• EEPROM memory storage
• Target comparison logic
• Motor control using relay
