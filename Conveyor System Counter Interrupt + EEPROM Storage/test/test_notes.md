TEST PROCEDURE
----------------

1. Power ON system.
2. LCD shows:
   Conveyor Counter

3. Press SET button:
   → Target value increases
   → Stored in EEPROM

4. Press START:
   → Relay ON
   → Counting begins

5. Trigger IR sensor manually:
   → Count increases

6. When count reaches target:
   → Relay OFF
   → Buzzer ON

7. Press RESET:
   → Count resets to 0
   → Buzzer OFF

8. Power cycle board:
   → Set value should remain saved.
