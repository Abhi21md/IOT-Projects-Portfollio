HARDWARE CONNECTION DETAILS
----------------------------

Arduino UNO / Nano

IR / Proximity Sensor:
--------------------------------
OUT → Pin 2 (Interrupt Pin)
VCC → 5V
GND → GND

Relay Module:
--------------------------------
IN → Pin 8
VCC → 5V
GND → GND

Buzzer:
--------------------------------
Positive → Pin 9
Negative → GND

Push Buttons:
--------------------------------
START  → Pin 3 (INPUT_PULLUP)
STOP   → Pin 4 (INPUT_PULLUP)
RESET  → Pin 5 (INPUT_PULLUP)
SET    → Pin 6 (INPUT_PULLUP)

LCD (I2C 16x2):
--------------------------------
SDA → A4
SCL → A5
VCC → 5V
GND → GND
I2C Address → 0x27
