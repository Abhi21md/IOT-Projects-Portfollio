# Hardware Components Used

----------------------------------
Microcontroller
----------------------------------
Arduino UNO

----------------------------------
RFID Module
----------------------------------
MFRC522

Connections:
SDA (SS)  → Pin 10
SCK       → Pin 13
MOSI      → Pin 11
MISO      → Pin 12
RST       → Pin 9
3.3V      → 3.3V
GND       → GND

----------------------------------
Servo Motor
----------------------------------
Signal → Pin 3
VCC → 5V
GND → GND

----------------------------------
LCD Display (I2C)
----------------------------------
SDA → A4
SCL → A5
VCC → 5V
GND → GND

----------------------------------
Other Components
----------------------------------
Green LED → Pin 5
Red LED   → Pin 6
Buzzer    → Pin 4

----------------------------------
Power Supply
----------------------------------
5V regulated power supply
