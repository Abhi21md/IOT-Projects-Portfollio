# RFID-Based Door Lock Security System

A secure embedded access control system using Arduino and MFRC522 RFID module.

--------------------------------------------------

## Features

• RFID-based authentication
• Servo controlled locking system
• LCD display status messages
• Buzzer alerts
• Green/Red LED indicators
• Automatic relock feature
• Expandable authorized card list

--------------------------------------------------

## Applications

• Home security
• Office access control
• Lab entry system
• Locker security
• Smart embedded access system

--------------------------------------------------

## Technology Stack

Microcontroller: Arduino UNO
Communication: SPI
Display: I2C LCD
Actuator: Servo Motor
