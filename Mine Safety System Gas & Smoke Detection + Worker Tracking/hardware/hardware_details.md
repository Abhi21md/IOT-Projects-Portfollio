HARDWARE CONNECTIONS
--------------------

ESP8266 (NodeMCU)

----------- MQ2 -----------
VCC → 5V
GND → GND
AO  → A0

----------- DHT11 ----------
VCC → 3.3V
GND → GND
DATA → D4

----------- RFID MFRC522 -----------
SDA → D8
RST → D0
MOSI → D7
MISO → D6
SCK → D5
3.3V → 3.3V
GND → GND

----------- ALERT ----------
Buzzer → D3
LED → D5

----------- LCD I2C ----------
SDA → D2
SCL → D1
VCC → 5V
GND → GND
