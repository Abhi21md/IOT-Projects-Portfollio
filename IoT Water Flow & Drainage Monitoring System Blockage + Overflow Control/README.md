# IoT-Based Water Flow & Drainage Monitoring System

An advanced IoT drainage automation system with intelligent flow monitoring,
auto pump control, and cloud alerts using Blynk.

--------------------------------------------------

## Key Highlights

• Real-time water flow measurement
• Total water usage logging
• Soil moisture based overflow detection
• Blockage detection with hysteresis
• Auto-unblock pump activation
• Manual override via Blynk
• Calibration factor adjustable from cloud
• Alert cooldown system
• Noise filtering algorithms
• Active LOW relay safe logic

--------------------------------------------------

## Applications

• Smart drainage systems
• Agriculture irrigation control
• Smart water management
• Basement overflow protection
• Industrial water flow monitoring

--------------------------------------------------

## Technology Stack

Microcontroller: ESP8266
Communication: WiFi
Cloud: Blynk IoT
Interrupt Handling: Hardware Interrupt
Filtering: Weighted Moving Average
Control Logic: Hysteresis Based Automation
