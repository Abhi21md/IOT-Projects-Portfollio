# Smart Plant Monitoring System (IoT Based)

A real-time plant monitoring and protection system
using NodeMCU, Blynk IoT, DHT11, Soil Sensor and PIR.

------------------------------------------------------

## Key Features

• Soil moisture monitoring
• Temperature & humidity tracking
• Motion detection near plant
• Relay control via cloud + physical button
• LCD live monitoring
• Cloud event alerts

------------------------------------------------------

## System Highlights

• Smart change-based cloud updates
• Remote PIR enable/disable
• Dual control architecture
• Stable timed sensor polling
• IoT + local display integration

------------------------------------------------------

## Technology Stack

NodeMCU (ESP8266)
Blynk IoT
DHT11
Soil Moisture Sensor
PIR Sensor
Relay Module
I2C LCD

------------------------------------------------------

Applications

• Smart indoor plant care
• Balcony garden monitoring
• Motion-based plant protection
• IoT agriculture mini project
